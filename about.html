About Stevenson

Stevenson is a micro-CMS leveraging GitHub for storing the website content and the Jekyll templating engine for templating and site configuration.

Manage Through GitHub

GitHub can manage the publishing and hosting of your content, Stevenson handles the editing.

No Coding Required

Set up a blog or website with no coding required.

Simple

Stevenson requires no configuration or backend coding, all you need is a GitHub account.
